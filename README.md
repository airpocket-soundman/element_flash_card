# element_flash_card

元素を覚えるためのシンプルなフラッシュカードWebアプリです。  
GitHub Pages でそのまま公開できる静的サイト構成です。

## 機能

- 4つの学習コース
  - 元素番号コース
  - 元素名(英語)コース
  - 元素名(日本語)コース
  - 元素記号コース
- 1問ごとに、選んだコースの項目だけ先に表示
- `答えを見る` で残り3項目を表示
- 自己採点で `正解` / `誤答` を記録して次の問題へ
- リザルト画面で各コースのワースト10（誤答率が高い順）を表示
- 履歴は各コースごとにローカル保存（過去10000問を保持）
- 表示集計は `10000 / 5000 / 1000 / 500` 問で切替可能
- すべての画面に `メニューに戻る` ボタンを配置

## ファイル構成

- `web/index.html`: 画面構成（メニュー / クイズ / リザルト）
- `web/styles.css`: スタイル
- `web/app.js`: 出題ロジック、履歴保存、リザルト集計
- `web/elements.js`: 元素データ（番号 / 英語名 / 日本語名 / 記号）

## 使い方（ローカル）

1. このリポジトリをダウンロードまたはクローン
2. `web/index.html` をブラウザで開く
3. メニューからコースを選択して開始
4. `答えを見る` -> `正解` または `誤答` を押して学習
5. `リザルトを見る` で弱点元素を確認

## GitHub Pages で配信する

`web/` 配下をそのまま公開したい場合は、GitHub Actions で `web` ディレクトリを Pages にデプロイしてください。  
`Deploy from a branch` の標準設定では `/ (root)` または `/docs` のみ指定できます。

## データ保存仕様

- 保存先: ブラウザの `localStorage`
- キー: `element_flash_card_history_v1`
- 保存単位: コース別履歴
- 1コースの保持上限: 10000件（超えた分は古い履歴から削除）

## 注意

- 正誤判定は自己採点方式です。
- `localStorage` を削除すると履歴も消えます。
